# Data Science Projects

A collection of notebooks covering statistical analysis, classification, and data visualization. The repository includes coursework and independent projects built around structured datasets and clear analytical workflows.

## Included Projects

### World Progress

Country-level analysis of population, fertility, child mortality, and poverty using joins, time series work, and comparative visualizations.

### Cardiovascular Disease

Statistical analysis of cardiovascular outcomes using simulation, hypothesis testing, relative risk, bootstrap intervals, and hazard-rate comparisons.

### Movie Classification

A text classification project that uses bag-of-words features and k-nearest neighbors to distinguish between screenplay genres.

## Repository Structure

- `project1 (2).ipynb` for World Progress
- `project2 (3).ipynb` for Cardiovascular Disease
- `project3 (1).ipynb` for Movie Classification

## Notes

These notebooks are intended to show applied work in data analysis, inference, and modeling rather than a single packaged software project.
